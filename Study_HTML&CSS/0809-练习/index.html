1、one和two均未设置高度，父级DIV设高度。故即使因line-height显示超出父级高度height，但占用高度仍是父级的高度！

DIV- height：50px; Span1- line-height: 50px; Span2- line-height: 98px;
我是第二个DIV！

2、父级DIV未设置高度，故父级高度由内容撑开。子级未设置高度，故子级高度height=行高line-height。进而父级高度=子级高度=子级行高

DIV- 无高 Span1- line-height: 50px; Span2- line-height: 98px;
我是第二个DIV！

3、父级DIV未设置高度，子级设置高度，且子级line-height>height。则子级占位为height，进而父级高度=子级高度

DIV- 无高 Span1- line-height: 50px; height: 18px; Span2- line-height: 98px; height: 12px;
我是第二个DIV！
